# Quincy User Guide

## 1️⃣ What is Quincy?

Quincy is an AI-powered Telegram assistant that helps you interact with the TON blockchain — check balances, track prices, and understand TON safely.

## 2️⃣ How to Start

1. Open Telegram
2. Search for @QuincyBot (or your bot)
3. Press Start

That’s it.

## 3️⃣ Main Things You Can Do

### 🔍 Check Wallet Balance
- Paste a TON wallet address
- Quincy shows:
  - TON balance
  - USDT (Jetton) balance
  - Recent transactions

### 📈 Track TON Price
- Ask: What is the TON price?
- Or use the dashboard buttons
- Set price alerts
- Get notified automatically

### 🧠 Ask TON Questions
- Example:
  - “What is a Jetton?”
  - “How do I secure my wallet?”
- Quincy gives clear, safe answers

### 🛡️ Stay Safe
- Quincy warns you if:
  - You try to share a seed phrase
  - A message looks like a scam

## 4️⃣ Tips for Best Results
- Never share your seed phrase
- Use price alerts instead of checking charts
- Ask questions in simple language

## 5️⃣ Who is Quincy for?
- TON beginners
- TON traders
- Telegram communities
- TON project admins
